Interface TextStyleParam

テキストの更新スタイル

It is based on [PIXI.TextStyle.constructor] option

このオブジェクトはPIXI.TextStyleを継承しない。 PIXI.TextStyleにはデフォルト値があるため、インスタンス化するとundefinedな値を受け付けない。 TextStyleParamでは、undefinedは更新以前のパラメーターを維持する。

see

https://pixijs.download/dev/docs/PIXI.TextStyle.html
